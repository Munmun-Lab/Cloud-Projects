# Host a Dynamic Web App on AWS

In this project, you'll learn how to host a dynamic website on AWS, using a hands-on approach. We'll cover essential AWS services, including VPC setup with public and private subnets, security groups, EC2 instances, NAT gateways, RDS, and Application Load Balancers. You'll also explore Route 53, Certificate Manager, and Auto Scaling groups. This practical experience is perfect for beginners, providing a foundational understanding of AWS website hosting.
